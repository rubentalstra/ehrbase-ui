# ADR-0003 — SSR mode (`ssr: 'data-only'` for authed routes)

- **Status:** Proposed
- **Date:** 2026-05-26

## Context

Stub. Full content lands with Milestone 2. Architecture-doc reference: §4 "SSR mode for this app".

## Decision

Use `ssr: 'data-only'` on every authenticated route; default SSR on public routes; full SSR on the OIDC callback. To be ratified during Milestone 2.
