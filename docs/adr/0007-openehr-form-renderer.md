# ADR-0007 — openEHR dynamic form renderer

- **Status:** Proposed
- **Date:** 2026-05-26

## Context

Stub. Full content lands with Milestone 5. Architecture-doc reference: §7.

The summary: EHRbase returns a JSON "web template" describing the form structure of any operational template. The UI compiles this into a working react-hook-form at runtime via a four-stage pipeline (web template → Zod schema → react-hook-form → shadcn Field renderer → FLAT converter).
